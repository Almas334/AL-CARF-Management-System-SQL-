The AL-CARF Management System is a database-driven project developed using SQL to efficiently manage and organize car rental operations. The system is designed to handle key functionalities such as customer information, vehicle details, booking records, and rental transactions.

This project includes structured database design with multiple tables, relationships, and constraints to ensure data integrity and consistency. SQL queries are used for data insertion, updating, deletion, and retrieval, enabling smooth management of rental activities.

Key features of the system include tracking available and rented vehicles, managing customer records, monitoring booking status, and generating useful insights through queries. The project demonstrates strong understanding of relational database concepts, normalization, and practical use of SQL for real-world applications.
